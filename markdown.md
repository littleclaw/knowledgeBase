+ 标题 使用#，##开头，类似html里的h1,h2标签
+ 列表 使用+、-、\*表示无序列表，用数字点加空格，如1. 4. 表示有序列表
+ 引用 用大于号>表示引用
+ 图片与链接 图片![label](url) 链接[label](url)
+ 粗体 斜体	用**引起来的文本为粗体**，用*引起来为斜体*
+ 代码 	用''引起来的文本为代码块
+ 分隔线  用三个\*表示分隔线
***
+ 页内跳转 链接后跟(#id),然后把要跳转的区域改写成html标签形式，在该标签上加上id,如

 + 传剑(#2)
